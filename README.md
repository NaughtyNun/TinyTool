# TinyTool
A weekend coding project, from something very simple (Split & Merge of PDF Documents from the command line) to building a GUI application with a search facility and then, because of way too many 
calls regarding Adobe Acrobat, the ability to generate Security Declarations to Time Sheets was developed. Having a more complete tool (and some calls regarding FTP) a FTP client was also built. 
TinyTool was mainly created to make ordinary tasks that was causing both frustration and time lost much easier and way faster.
